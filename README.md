# Insurance Claim Approval Prediction

## 📌 Project Overview

This project aims to predict whether an insurance claim will be approved or denied using Machine Learning techniques. By analyzing historical claim-related data, the model helps automate the claim evaluation process and supports faster decision-making.

---

## 🎯 Objective

Develop a classification model that can accurately predict the status of an insurance claim based on various customer, policy, and claim-related features.

---

## 📊 Dataset

The dataset contains insurance claim records along with multiple features related to:

- Customer information
- Policy details
- Claim information
- Employee/Vendor information (if applicable)
- Claim approval status (Target Variable)

### Target Variable
- Approved
- Denied
